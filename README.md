# websocmensagem

Um estudo de **Spring Security** e uso de **WebSocket**.

---

## Como rodar

### Backend
1. Criar as Keys JWT na pasta `resources` do projeto.
2. Configurar o arquivo `application.properties`.
3. Rodar o comando:
   `mvn spring-boot:run`

### TUI
1. Entrar na pasta do `TUI` e descompactar o arquivo `BlessedTUI`.
2. Entrar na pasta descompactada e rodar:
   `npm install && node index.js`

---
